mobileNet-webcam






                  # 사용법
                    1. webcam 활성화.

                    2. classA, classB, classC에 원하는 클래스 이름을 적고 '설정하기'를 누른다.
                    ㄴ클래스 이름을 다 적고, 설정하기를 누른다음 Add버튼을 눌러야한다.
                    ㄴ이를 따르지 않으면 class이름이 제대로 설정되지 않는다.
                    
                    3. 웹캠에 인식할 물체를 보여주고 A B C 버튼중 하나를 누른다
                    ㄴex) 스마트폰을 A object라고 가정하고, 웹캠에 스마트폰을 보이며 Add A 버튼 최소 10회이상 클릭
                    ㄴ 그리고 사람(본인)을 웹캠에 보이며 Add B 버튼을 10회이상 클릭
                    3. 웹 캠에 인식하고자 하는 물체(2번에서 미리 학습한 물체)를 보인다.
